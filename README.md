# Apex Waters

The Great White is lurking...

## Features
- Custom Great White Shark entity that naturally spawns across all ocean biomes
- Dynamic hunger system — hunger initializes at a random level and depletes over time, triggering a deadly hunting mode when it hits zero
- Retaliates when attacked — don't pick a fight you can't finish
- Blood scent mechanic — the shark becomes aggressive when a nearby player is low on health, just like the real thing

## Configuration
A built-in config file lets you fine-tune the experience to your liking — adjust HP, toggle whether the shark targets players in boats, and much more.

## Compatibility
Designed to work seamlessly with any mod pack. Compatible with all custom mobs and biomes out of the box.
